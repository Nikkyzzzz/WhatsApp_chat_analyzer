Input your exported WhatsApp chat in the "WhatsappChat to csv.py" to make a cleaned .csv file from raw .txt file. Use the .csv file as input for "ChatAnalyzer.py" to get your chat analyzed on various metrics. It also has integrated Cohere chat model to ask questions about the chat data and to get certain insights

Remember to replace your trial cohere api keys with "your-api-keys-here". The keys are free and can be generated on cohere website. Sign-up for Cohere and get your free keys generated. After signing up for Cohere go to https://dashboard.cohere.com/api-keys adn generate a trial key.
